Audit Utility
  • Summary Demo Technology Involved History

    A third party application for Microsoft Dynamics SL that creates audits for any database table in the currently selected company. The utility offers automatic creation, manually copying the SQL script and deletion of any audits created by the utility.

    • Microsoft Dynamics SL
    • mySQL
    • VBA

    While at Solomon Cloud Solutions, the idea was proposed as something that would benefit customers in need of constant audit scripts. Creating these scripts are trivial but rather tedious so instead of taking the time to create them, why not have an application that can solve most of these needs?
